### （2）解决鸡兔同笼问题

Python

```Python
n = int(input())
m = int(input())
a = m/2 - n
b = n - a
if a%1 == 0 and b%1 == 0 and  a >= 0 and  b >= 0:    
    print("鸡的数量%d" %(b))   
    print("兔的数量%d" %(a))
else:   
    print("无解")
```
